# 🧠 HR Workflow Designer

An interactive web application to design and simulate HR workflows using a visual drag-and-drop interface.

---

## 🚀 Overview

HR Workflow Designer allows users to create structured workflows by connecting different types of nodes like Start, Task, Approval, and End. It visually represents the process flow and simulates execution.

---

## ✨ Features

- 🟢 Add different node types (Start, Task, Approval, End)
- 🔗 Connect nodes to form workflows
- ✏️ Edit node details dynamically
- 🗑️ Delete unwanted nodes
- ▶️ Run workflow simulation
- 🎯 Visual representation of process flow
- ⚡ Fast and lightweight UI
